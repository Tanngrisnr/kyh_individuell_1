Website appearance

Inspiration

The visual design of this website was in large part inspired by this blank video cassette tape, from it I extracted most of the colors used in this website, lightening the cream color to better fit as a bakground color and making minor alterations to the grey. Making use of the skew-property of the CSS transform-property the website typography and layout is transformed as inspired by the text and imagery of the cassette tape cover.

Layout

The CSS is built to be response making use of no absolute values when it comes to measurments of the site and instead relying on relative measurements based on widht of parent element and viewport height/width to govern the size of html elements. The site navigation utilzes both relative size and skewing to achieve an apperance similar to the rainbow present on the inspiration image as well as animations to act as a dynamic menu.

The site also utilizes flexbox to create a responsive layout. For the article elements a "row" flex-direction is used above certain screen sizes, where instead the "collumn" flex-direction is used.
